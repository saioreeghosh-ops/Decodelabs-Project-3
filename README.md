# Decodelabs-Project-3
# SQL Data Analysis

## Overview
This project is part of the DecodeLabs Industrial Training Kit (Batch 2026).  
The goal of Project 3 is to demonstrate SQL fundamentals by querying a cleaned dataset and extracting actionable business insights.

## Objectives
- Write `SELECT` queries
- Use `WHERE`, `ORDER BY`, and `GROUP BY`
- Perform basic aggregations (`COUNT`, `SUM`, `AVG`)
- Apply SQL to generate meaningful business intelligence

## Dataset
The dataset (`cleaned_data.xlsx`) contains e‑commerce order records with fields such as:
- OrderID, CustomerID, Product, Quantity, UnitPrice, TotalPrice
- PaymentMethod, OrderStatus, CouponCode, ReferralSource

## Queries Implemented
1. **SELECT** – Extract specific columns and preview data  
2. **WHERE** – Filter rows by conditions (e.g., shipped orders, high‑value orders)  
3. **GROUP BY** – Categorize data into buckets (e.g., orders per product, per payment method)  
4. **Aggregations** – Calculate totals and averages (e.g., revenue, average spend)  
5. **ORDER BY** – Sort results (e.g., top customers by spend)

## Business Insights
- **Top Products by Revenue** – Identify which products generate the most sales  
- **Most Popular Coupon Codes** – Evaluate promotion effectiveness  
- **Revenue by Referral Source** – Understand which marketing channels drive revenue  
- **Average Order Value by Payment Method** – Compare customer spending behavior  
- **Order Status Distribution** – Assess operational performance

## How to Run
1. Open the Jupyter Notebook (`Project3_SQL_Data_Analysis.ipynb`).  
2. Ensure Python libraries `pandas` and `pandasql` are installed.  
3. Load the dataset (`cleaned_data.xlsx`).  
4. Run cells step by step to reproduce the analysis.  

## Conclusion
This project demonstrates the ability to query structured data using SQL and transform raw records into insights.  
By completing Project 3, the foundation for advanced analytics and reporting pipelines is established.
